# 🗂️ script-sh-collection

> A curated collection of useful shell scripts for dev environment setup, automation, and daily workflows.

## 🚀 Usage

```bash
# Add permission
chmod +x <script-name>.sh

# Run
./<script-name>.sh
```

## ⚠️ Disclaimer

Always review a script before running it. Some scripts may modify system configurations (e.g., `.zshrc`, `.bashrc`).
